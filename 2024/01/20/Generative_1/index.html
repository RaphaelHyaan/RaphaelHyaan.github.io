Chapter 1 Generative Modeling

Generative modeling is a branch of machine learning that involves training a model to produce new data that is similar to a given dataset.

жҰӮеҝө

ж·ұеәҰеӯҰд№ жЁЎеһӢ

д»Ҙз”ҹжҲҗдёҺжҸҗдҫӣзҡ„ж•°жҚ®йӣҶзӣёдјјзҡ„ж–°ж•°жҚ®дёәзӣ®зҡ„зҡ„жңәжў°еӯҰд№ еҲҶж”ҜгҖӮ

  • и®ӯз»ғж•°жҚ®еҢ…еҗ«еҫҲеӨҡдҫӢеӯҗпјҢе…¶дёӯжҜҸдёҖдёӘж•°жҚ®зӮ№иў«з§°дёәдёҖдёӘи§ӮеҜҹ observation
  • жҜҸдёӘи§ӮеҜҹеҢ…еҗ«еҫҲеӨҡзү№еҫҒ feature
  • жҲ‘д»¬зҡ„зӣ®ж ҮжҳҜжһ„е»әдёҖдёӘжЁЎеһӢпјҢиҜҘжЁЎеһӢеҸҜд»Ҙз”ҹжҲҗж–°зҡ„зү№еҫҒйӣҶпјҢиҝҷдәӣзү№еҫҒйӣҶзңӢиө·жқҘе°ұеғҸжҳҜдҪҝз”ЁдёҺеҺҹе§Ӣж•°жҚ®зӣёеҗҢзҡ„и§„еҲҷеҲӣе»әзҡ„гҖӮ
  • з”ҹжҲҗејҸжЁЎеһӢеә”иҜҘжҳҜжҰӮзҺҮзҡ„probabilistic иҖҢдёҚжҳҜзЎ®е®ҡзҡ„deterministic

дёҺеҲӨеҲ«ејҸжЁЎеһӢзҡ„еҢәеҲ«

  • еҲӨеҲ«ејҸжЁЎеһӢйў„жөӢ$p(y \mid x)$
  • з”ҹжҲҗејҸжЁЎеһӢйў„жөӢ$p(x)$пјҢжҲ–иҖ…еңЁдёҖе®ҡжғ…еҶөдёӢйў„жөӢ$p(x|y)$
    • For example, if our dataset contains different types of fruit, we could tell our generative model to specifically generate an image of an apple.

з”ҹжҲҗејҸжЁЎеһӢжҳҜи§Јй”ҒдёҖз§ҚжӣҙеӨҚжқӮзҡ„дәәе·ҘжҷәиғҪеҪўејҸзҡ„е…ій”®

  • йҰ–е…ҲпјҢзәҜзІ№д»ҺзҗҶи®әи§’еәҰжқҘзңӢпјҢжҲ‘д»¬дёҚеә”иҜҘе°ҶжңәеҷЁи®ӯз»ғйҷҗеҲ¶еңЁз®ҖеҚ•зҡ„ж•°жҚ®еҲҶзұ»дёҠгҖӮдёәдәҶе®Ңж•ҙжҖ§пјҢжҲ‘д»¬иҝҳеә”иҜҘе…іжіЁи®ӯз»ғжЁЎеһӢпјҢд»ҘиҺ·еҸ–еҜ№ж•°жҚ®еҲҶеёғзҡ„жӣҙе®Ңж•ҙзҡ„зҗҶи§ЈпјҢиҖҢдёҚжҳҜд»»дҪ•зү№е®ҡзҡ„ж Үзӯҫ
  • е…¶ж¬ЎпјҢз”ҹжҲҗе»әжЁЎзҺ°еңЁжӯЈиў«з”ЁдәҺжҺЁеҠЁдәәе·ҘжҷәиғҪе…¶д»–йўҶеҹҹзҡ„иҝӣжӯҘ
  • жңҖеҗҺпјҢеҰӮжһңжҲ‘д»¬зңҹзҡ„иҰҒиҜҙжҲ‘д»¬е·Із»Ҹе»әйҖ дәҶдёҖеҸ°жңәеҷЁпјҢе®ғиҺ·еҫ—дәҶдёҺдәәзұ»зӣёеҪ“зҡ„жҷәиғҪеҪўејҸпјҢйӮЈд№Ҳз”ҹжҲҗе»әжЁЎиӮҜе®ҡжҳҜи§ЈеҶіж–№жЎҲзҡ„дёҖйғЁеҲҶ

жҲ‘д»¬зҡ„з¬¬дёҖдёӘз”ҹжҲҗејҸжЁЎеһӢ

ж„ҹзҹҘж•°жҚ®з©әй—ҙ

  • и§ӮеҜҹзҡ„йӣҶеҗҲпјҡзӮ№
  • жҲ‘д»¬еҒҮи®ҫиҝҷдәӣи§ӮеҜҹз”ҹжҲҗдәҺдёҖдёӘжңӘзҹҘзҡ„еҲҶеёғ$p_{data}$
  • жһ„е»әдёҖдёӘз”ҹжҲҗејҸжЁЎеһӢ$p{model}$пјҢдҪҝеҫ—иҝҷдёӘжЁЎеһӢз”ҹжҲҗзҡ„ж•°жҚ®зңӢиө·жқҘжҳҜ$p{data}$з”ҹжҲҗзҡ„

ж•°жҚ®з©әй—ҙйҮҚеҗҲ еҰӮдёҠеӣҫпјҢжҲ‘д»¬и§ӮеҜҹиҝҷдәӣзӮ№пјҢеҸҜиғҪеҫ—еҲ°з¬¬дәҢе№…еӣҫдёӯж©ҷиүІзҡ„еҢәеҹҹгҖӮжҢүз…§иҝҷдёӘеҢәеҹҹжҲ‘д»¬йҡҸжңәз”ҹжҲҗдёҖдёӘзәўиүІзҡ„зӮ№AгҖӮдҪҶе®һйҷ…зҡ„еҲҶеёғжҳҜд»…д»…з”ҹжҲҗеңЁеӨ§йҷҶдёҠпјҢжүҖд»ҘжҲ‘д»¬з”ҹжҲҗзҡ„ж•°жҚ®е№¶жІЎжңүзңӢиө·жқҘжҳҜ$p{data}$з”ҹжҲҗзҡ„ж•°жҚ®гҖӮиҝҷжҳҜз”ҹжҲҗжЁЎеһӢзҡ„з©әй—ҙеҢ…еҗ«дәҶеҺҹж•°жҚ®з©әй—ҙдёҚеҢ…жӢ¬зҡ„еҢәеҹҹгҖӮиҖҢзәўиүІзҡ„зӮ№BеҲҷеңЁж©ҷиүІеҢәеҹҹд№ӢеӨ–пјҢдёҚеҸҜиғҪз”ұ$p{model}$з”ҹжҲҗпјҢиҝҷжҳҜз”ҹжҲҗжЁЎеһӢжІЎжңүеҢ…еҗ«еә”иҜҘеҢ…еҗ«зҡ„еҢәеҹҹгҖӮ

е®№жҳ“з”ҹжҲҗ е°Ҫз®ЎжңүзјәзӮ№пјҢдҪҶиҜҘжЁЎеһӢеҫҲе®№жҳ“йҮҮж ·пјҢеӣ дёәе®ғеҸӘжҳҜж©ҷиүІзӣ’еӯҗдёҠзҡ„еҚ•дёҖеҲҶеёғгҖӮ

иЎЁеҫҒеӯҰд№  representation learning

жҲ‘д»¬дёҚиҜ•еӣҫзӣҙжҺҘеҜ№й«ҳз»ҙж ·жң¬з©әй—ҙиҝӣиЎҢе»әжЁЎпјҢиҖҢжҳҜдҪҝз”ЁдёҖдәӣдҪҺз»ҙжҪңеңЁз©әй—ҙжқҘжҸҸиҝ°и®ӯз»ғйӣҶдёӯзҡ„жҜҸдёӘи§ӮжөӢпјҢз„¶еҗҺеӯҰд№ дёҖдёӘжҳ е°„еҮҪж•°пјҢиҜҘеҮҪж•°еҸҜд»ҘиҺ·еҸ–жҪңеңЁз©әй—ҙдёӯзҡ„дёҖдёӘзӮ№пјҢе№¶е°Ҷе…¶жҳ е°„еҲ°еҺҹе§Ӣеҹҹдёӯзҡ„дёҖзӮ№гҖӮ

е…·дҪ“жқҘиҜҙпјҢеҰӮдёҠиҝ°10дёӘеӣҫеҪўдёӯпјҢе°Ҫз®Ўд»–д»¬йғҪжҳҜ3з»ҙзҡ„еӣҫеҪўпјҢдҪҶжҲ‘д»¬зҹҘйҒ“пјҢжҲ‘д»¬еҸӘйңҖиҰҒзҹҘйҒ“еә•йқўеҚҠеҫ„е’Ңй«ҳеҚіеҸҜгҖӮд»–еӣ жӯӨжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮйҖӮеҪ“зҡ„жҳ е°„еҮҪж•°пјҢд»ҺдёҖдёӘдәҢз»ҙзҡ„жҪңеңЁз©әй—ҙз”ЁдәҺз”ҹжҲҗж–°зҡ„еңҶжҹұдҪ“гҖӮ

ж ёеҝғжҰӮзҺҮи®ә

е…ій”®жңҜиҜӯ

  • ж ·жң¬з©әй—ҙпјҡжүҖжңүеҸҜиғҪзҡ„и§ӮжөӢеҸҜеҸ–зҡ„еҖјгҖӮжҜ”еҰӮд№ӢеүҚдё–з•Ңең°еӣҫзҡ„дҫӢеӯҗпјҢжүҖжңүз»Ҹзә¬еәҰеҜ№еә”зҡ„зӮ№йғҪжҳҜж ·жң¬з©әй—ҙзҡ„зӮ№пјҢж— и®әе®ғжҳҜеҗҰжҳҜзңҹжҳҜзҡ„ж•°жҚ®еҲҶеёғдёӯзҡ„дёҖдёӘзӮ№гҖӮ
  • жҰӮзҺҮеҜҶеәҰеҮҪж•°пјҡеҰӮеҗҚпјӣеңЁд№ӢеүҚзҡ„дҫӢеӯҗдёӯпјҢжүҖжңүж©ҷиүІжЎҶдёӯзҡ„жҰӮзҺҮеҜҶеәҰеҮҪж•°зҡ„з§ҜеҲҶдёә1пјҢжЎҶеӨ–зҡ„жҰӮзҺҮеҜҶеәҰеҮҪж•°дёә0
  • еҸӮж•°еҢ–е»әжЁЎпјҡеҸӮж•°еҢ–е»әжЁЎжҳҜдёҖз§ҚйҖҡиҝҮйғЁеҲҶеҸӮж•°жқҘеҜ№$p{model}$е»әжЁЎпјҢеңЁиҝҷз§Қе»әжЁЎж–№жі•дёӢпјҢеӯҳеңЁеҸӮж•°йӣҶеҗҲ$\theta$пјҢдҪҝеҫ—$p{\theta}(x) = p{model}(x)$гҖӮеңЁе…ҲеүҚзҡ„дҫӢеӯҗдёӯпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”ЁеӣӣдёӘеқҗж ҮжҸҸиҝ°ж©ҷиүІжЎҶпјҢеҚі$p{(x1,x_2,x_3,x_4)}(x) = p{model}(x)$
  • дјјз„¶Likelihoodпјҡдјјз„¶зҡ„е®ҡд№үжҳҜеңЁеҪ“еүҚеҸӮж•°йӣҶ$\theta$дёӢпјҢи§ӮжөӢзӮ№xзҡ„жҰӮзҺҮеҜҶеәҰгҖӮ$\mathscr{L}(\theta \mid \mathbf{x})=p\theta(\mathbf{x})$гҖӮеҜ№дәҺи§ӮжөӢзӮ№зҡ„йӣҶеҗҲXпјҢе…¶дјјз„¶е®ҡд№үдёәпјҡ$\mathscr{L}(\theta \mid \mathbf{X})=\prod{\mathbf{x} \in \mathbf{X}} p_\theta(\mathbf{x})$гҖӮ
    • еңЁе…ҲеүҚзҡ„дҫӢеӯҗдёӯпјҢдёҖдёӘеҸӘиҰҶзӣ–ең°еӣҫе·ҰеҚҠйғЁеҲҶзҡ„ж©ҷиүІж–№жЎҶзҡ„еҸҜиғҪжҖ§дёә0вҖ”вҖ”е®ғдёҚеҸҜиғҪз”ҹжҲҗж•°жҚ®йӣҶпјҢеӣ дёәжҲ‘д»¬еңЁең°еӣҫеҸіеҚҠйғЁеҲҶи§ӮеҜҹеҲ°дәҶзӮ№пјҢиҝҷдәӣзӮ№зҡ„жҰӮзҺҮдёә0гҖӮ
    • дёәдәҶж–№дҫҝи®Ўз®—пјҢжҲ‘д»¬дҪҝз”ЁеҜ№ж•°дјјз„¶пјҡ$\ell(\theta \mid \mathbf{X})=\sum{\mathbf{x} \in \mathbf{X}} \log p\theta(\mathbf{x})$
    • жңҖйҮҚиҰҒзҡ„жҳҜпјҢ**дјјз„¶жҳҜеҸӮж•°зҡ„еҮҪж•°иҖҢйқһи§ӮеҜҹзҡ„еҮҪж•°**
    • еҸӮж•°еҢ–е»әжЁЎзҡ„йҮҚзӮ№еҚідёәжңҖеӨ§еҢ–дјјз„¶еҮҪж•°
  • **жңҖеӨ§дјјз„¶дј°и®Ўпјҡ**еҰӮдёҠжүҖеұһпјҢеә”жңҖеӨ§еҢ–дјјз„¶гҖӮжңҖеӨ§дјјз„¶иў«е®ҡд№үдёәпјҡ$\hat \theta = argmax_{x}\ell(\theta|X)$
    • еңЁдёҠдҫӢдёӯпјҢжңҖеӨ§дјјз„¶дј°и®ЎжҳҜеҢ…еҗ«жүҖжңүзӮ№зҡ„жңҖе°Ҹзҹ©йҳө
    • жңҖе°ҸеҢ–иҙҹеҜ№ж•°дјјз„¶еҮҪж•°пјҡ$\hat \theta = argmin_{x}(-\ell(\theta|X))$
    • еҪ“з„¶пјҢе®һйҷ…дёҠеңЁж·ұеәҰеӯҰд№ дёӯпјҢдёҚеҸҜиғҪзӣҙжҺҘи®Ўз®—жңҖе°Ҹзҡ„дјјз„¶еҮҪж•°

з”ҹжҲҗејҸжЁЎеһӢеҲҶзұ»

ж №жҚ®жЁЎеһӢзҡ„е»әз«Ӣж–№ејҸпјҢеӨ§иҮҙжңүдёүдёӘеӨ§зұ»пјҡ

  • еҜ№еҜҶеәҰеҮҪж•°иҝӣиЎҢжҳҫејҸе»әжЁЎпјҢдҪҶд»Ҙжҹҗз§Қж–№ејҸзәҰжқҹжЁЎеһӢ
    • еҸҜеӨ„зҗҶжЁЎеһӢеҜ№жЁЎеһӢжһ¶жһ„ж–ҪеҠ дәҶзәҰжқҹпјҢдҪҝеҫ—еҜҶеәҰеҮҪж•°е…·жңүжҳ“дәҺи®Ўз®—зҡ„еҪўејҸгҖӮдҫӢеҰӮпјҢиҮӘеӣһеҪ’жЁЎеһӢеҜ№иҫ“е…Ҙзү№еҫҒиҝӣиЎҢжҺ’еәҸпјҢд»ҘдҫҝеҸҜд»ҘжҢүйЎәеәҸз”ҹжҲҗиҫ“еҮәвҖ”вҖ”дҫӢеҰӮпјҢйҖҗдёӘеҚ•иҜҚжҲ–йҖҗдёӘеғҸзҙ гҖӮеҪ’дёҖеҢ–жөҒжЁЎеһӢе°ҶдёҖзі»еҲ—жҳ“дәҺеӨ„зҗҶзҡ„еҸҜйҖҶеҮҪж•°еә”з”ЁдәҺз®ҖеҚ•еҲҶеёғпјҢд»Ҙз”ҹжҲҗжӣҙеӨҚжқӮзҡ„еҲҶеёғ
  • еҜ№еҜҶеәҰеҮҪж•°зҡ„жҳ“дәҺеӨ„зҗҶзҡ„иҝ‘дјјиҝӣиЎҢжҳҫејҸе»әжЁЎ
    • иҝ‘дјјеҜҶеәҰжЁЎеһӢеҢ…жӢ¬еҸҳеҲҶиҮӘеҠЁзј–з ҒеҷЁпјҢе®ғеј•е…ҘжҪңеңЁеҸҳйҮҸе№¶дјҳеҢ–иҒ”еҗҲеҜҶеәҰеҮҪж•°зҡ„иҝ‘дјјгҖӮеҹәдәҺиғҪйҮҸзҡ„жЁЎеһӢд№ҹеҲ©з”Ёиҝ‘дјјж–№жі•пјҢдҪҶйҖҡиҝҮй©¬е°”еҸҜеӨ«й“ҫйҮҮж ·иҖҢдёҚжҳҜеҸҳеҲҶж–№жі•жқҘе®һзҺ°гҖӮжү©ж•ЈжЁЎеһӢйҖҡиҝҮи®ӯз»ғжЁЎеһӢйҖҗжёҗеҜ№е…ҲеүҚе·ІжҚҹеқҸзҡ„з»ҷе®ҡеӣҫеғҸиҝӣиЎҢеҺ»еҷӘжқҘиҝ‘дјјеҜҶеәҰеҮҪж•°гҖӮ
  • йҖҡиҝҮзӣҙжҺҘз”ҹжҲҗж•°жҚ®зҡ„йҡҸжңәиҝҮзЁӢеҜ№еҜҶеәҰеҮҪж•°иҝӣиЎҢйҡҗејҸе»әжЁЎ
    • йҡҗејҸеҜҶеәҰжЁЎеһӢж №жң¬дёҚж—ЁеңЁдј°и®ЎжҰӮзҺҮеҜҶеәҰпјҢиҖҢжҳҜд»…дё“жіЁдәҺдә§з”ҹзӣҙжҺҘз”ҹжҲҗж•°жҚ®зҡ„йҡҸжңәиҝҮзЁӢгҖӮйҡҗејҸз”ҹжҲҗжЁЎеһӢжңҖи‘—еҗҚзҡ„дҫӢеӯҗжҳҜз”ҹжҲҗеҜ№жҠ—зҪ‘з»ңгҖӮ
