正则表达式语法

在 Java 中，\ 表示：我要插入一个正则表达式的反斜线，所以其后的字符具有特殊的意义。

  • \\ 转义
  • 位置
    • ^ 字符串开头
    • $ 字符串结尾
  • 匹配
    • * 0次或多次匹配前面的字符，注意zo*可以匹配z
    • + 1次或多次匹配前面的字符
    • ? 0次或1次匹配前面的字符
    • {n}匹配n次
    • {n,}至少匹配n次
    • {n,m}匹配n到m次，包含
    • 在匹配符之后再增加一个? ，意为非贪心的，匹配最小字符
    • . 匹配除\r\n之外的所有字符
    • x|y匹配x或y
    • [xyz] 匹配中括号中任意一个字符
    • [^xyz] 匹配不在括号中的任意一个字符
    • [a-z] 匹配a到z的任意一个小写字母
    • \\b 匹配边界字符，比如never的er，而非verb的er
    • \\B 与\\b相反，匹配非边界字符
    • \\d 匹配数字字符，等价于[0-9]
    • \\D 匹配非数字字符
    • \\s 匹配任意空白字符，等价与[\f \n \t \v]
    • \\S 与\\s相反

使用正则表达式

Pattern

  • 初始化正则表达式字符串 private static final String REGEX = "\\bcat\\b";
  • 创建正则表达式 Pattern p = Pattern.compile(REGEX);

Matcher 类

  • 初始化目标字符串 private static final String INPUT = "cat cat cat cattie cat";
  • **创建matcher对象 Matcher m = p.matcher(INPUT); // 获取 matcher 对象**

Matcher 方法

  • 查找方法
    • public boolean lookingAt() 尝试将从区域开头开始的输入序列与该模式匹配。
    • **public boolean find() 尝试查找与该模式匹配的输入序列的下一个子序列。**
    • public boolean find(int start）重置此匹配器，然后尝试查找匹配该模式、从指定索引开始的输入序列的下一个子序列。
  • 索引方法
    • public int start() 返回以前匹配的初始索引。
    • public int end() 返回最后匹配字符之后的偏移量。
  • 替换方法
    • public String replaceAll(String replacement) 替换模式与给定替换字符串相匹配的输入序列的每个子序列。
    • **public String replaceFirst(String replacement) 替换模式与给定替换字符串匹配的输入序列的第一个子序列。**
