基本要求、对象和类、包和import

发表于2023-11-19|更新于2023-11-22|课程笔记-java
|阅读量:

HelloWorld

  • public 访问修饰符
  • static 关键字
  • void 返回类型
  • main 方法名
  • String[] String类 输入值 字符串数组

基本要求

  • 大小写敏感
  • 类名首字母大写，使用驼峰式命名法，每个单词首字母大写
  • 方法名以小写字母开头，若包含多个单词，每个单词首字母大写
  • 源文件名必须与类名相同，以.java后缀
  • **************主方法入口**************为: public static void main(String[] args)
  • 注释：
    • //这是单行注释
    • /* */这是多行注释
    • /** */这是文档注释，在文档注释中，通常包含一些特定的标签，如 @param 用于描述方法参数，@return 用于描述返回值，@throws 用于描述可能抛出的异常等等，参考Java 文档注释
  • 源文件声明规则
    1. 一个源文件中只有一个public类，源文件的名称应该和 public 类的类名保持一致
    2. 一个源文件中可以有多个非public类
    3. 如果一个类定义在某个包中，那么 package 语句应该在源文件的首行
    4. 如果源文件包含 import 语句，那么应该放在 package 语句和类定义之间。如果没有 package 语句，那么 import 语句应该在源文件中最前面
    5. import 语句和 package 语句对源文件中定义的所有类都有效。在同一源文件中，不能给不同的类不同的包声明

Java 对象和类

  • 对象是类的实例，类是一个模板，描述一类对象的行为和状态

1. 构造方法

  • 每个类都有一个同名的构造方法，一个类可以有多个构造方法

2. 创建对象

  • 声明：需要提前声明对象
  • 实例化： 使用new关键字来创建一个对象
  • 初始化：创建对象时，会调用构造方法初始化对象

3. 访问实例变量和方法

  • 实例化对象：Puppy myPuppy = new Puppy( "tommy" );
  • 访问变量：myPuppy.puppyAge；
  • 使用方法：myPuppy.setAge( 2 );

包和import

  • 在 Java 中，如果给出一个完整的限定名，包括包名、类名，那么 Java 编译器就可以很容易地定位到源代码或者类。
  • import 语句就是用来提供一个合理的路径，使得编译器可以找到某个类
  • package语句可以用来定位当前文件夹，或者说这个文件被定义在了包里
  • 可以使用工程中中所有的Public文件，如JAVA文件夹中的base/ExEmployee/EmployeeTest.java文件可以直接访问base/HelloWorld.java中的public类，当然需要正确的package
